# Aplica la IA

Ecosistema de herramientas de productividad desarrolladas con IA y lenguaje natural (Vibecoding).

### Enfoque técnico
- **Ejecución Local:** Prioridad a la privacidad y soberanía de datos mediante el uso de modelos locales.
- **Vibecoding:** Desarrollo de software acelerado mediante lenguaje natural.
- **Automatización:** Optimización de flujos de trabajo creativos y técnicos.

### Proyectos principales
- **[aia-studio-lite](https://github.com/aplica-la-ia/aia-studio-lite):** Toolkit de preproducción de guion y vídeo con exportación a DaVinci Resolve.
- **[aia-self-teleprompter](https://github.com/aplica-la-ia/aia-self-teleprompter):** Aplicación Web Progresiva (PWA) offline para lectura segmentada de guiones.
- **[aia-dev-caddy](https://github.com/aplica-la-ia/aia-dev-caddy):** Dashboard para la gestión de comandos CLI y prompts para LLMs.
- **[aia-scribe](https://github.com/aplica-la-ia/aia-scribe):** Aplicación nativa (PySide6) para transcripción de audio automatizada con aceleración GPU (faster-whisper).
